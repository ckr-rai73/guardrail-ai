# Guardrail.ai Community Edition

Welcome to the open-source Community Edition of Guardrail.ai. This platform provides foundational AI governance and security controls for developers and researchers.

## Features
- **Veto Protocol**: Human-in-the-loop and autonomous kill-switches for rogue AI actions.
- **Shadow Model Auditor**: Twin-LLM verification of intent.
- **Vector Clock Ledger**: Immutable audit trails.
- **Policy Engine**: Centralized rules engine for AI scopes.

## Enterprise Features
The Community Edition is feature-limited. The following capabilities require an upgrade to Guardrail.ai Enterprise:
- Insurance Oracle Risk Scoring
- Multi-Jurisdiction Skill Packs
- Red-Team as a Service (RT-RTaaS)
- Cloud Marketplace Connectors (AWS, Azure, GCP)
- Underwriting Gateway
